POS-Point of Sales Product Stock Odoo Apps

This Odoo apps is used to display stock in POS interface also the seller can select and display different type of stock like "Qty on hand", "Incoming Qty", "Outgoing Qty", "Qty Available" in POS and Allow/Deny POS sale Order when product is out of stock. You can display stock of All & Specific stock location in POS. Also Stock will be automatically sync when order get placed in Odoo POS inventory management App.

Features

Stock for POS Screen.

Seller can select and display different stock type like "Qty on hand", "Incoming Qty", "Outgoing Qty", "Qty Available" in POS.

Shows error in as per the Stock Availability.

Display of Different type of Error messages in POS interface based on stock availability.

Gives allowance as well as denial.

Seller can set the Unit, Deny order when quantity goes down to set unit. Allow/Deny POS sale Order when product is out of stock.

Display stock as per the user demand.

Show stock of all stock location and show stock of only specific stock location on POS session.

Stock of all the location in POS.

Show stock of all stock location and show stock of only specific stock location on POS session.

Sync the stock.

Stock will be automatically sync when order get placed in POS.

POS Stock Configuration

Under Point of Sale > Configuration > Point of Sale user have to enable "Display Stock in POS", Select "Stock Type", enable Allow POS Order When Product is Out of Stock", and enter "Deny pos order" if user wants to deny.

User also have to select "Show Stock Of" and select stock location for product.

Product On Hand Quantity on Backend

User can see total available onhand quantity for product.

Product On Hand Quantity on POS

User can see product onhand quantity on pos screen.

If user add quantity more than available quantity in orderline for a particular product then stock qty validation will be added.

If user select "Current Session Warehouse" then selected stock locations product stock will displayed on POS.

Product Qty by Stock Locations

User can see selected stock locations stock is displayed in pos.

User also click on "Sync" button, on clicking product quantity will updated.

Automatically update the product quantity if pos order is created.

Stock Will be automatically updated on POS product screen, no need to reload the POS screen everytime after you placed an order.

Stock quantity validation error will occur if the stock is lower than the deny amount.

Stock quantity validation error will occur if the product is out of stock and allow pos checkbox is disabled.

If user add quantity more than available quantity in current session warehouse in orderline for a particular product then stock qty validation will occur.

Stock quantity validation will not be applied for "Consumable" or "Service" product. Although user can add more qty in the orderline, it will go to the payment screen without validation.
